<h1>TriPeaks AI</h1>

Questo progetto implementa un'intelligenza artificiale per giocare a TriPeaks Solitaire.

<h3>Regole del gioco</h3>

TriPeaks è un gioco di carte solitario giocato con un mazzo standard da 52 carte.

<h3>Obiettivo</h3>

L'obiettivo è rimuovere tutte le carte dai tre picchi.

<h3>Setup</h1>

Le carte sono disposte in tre piramidi (peaks).

Solo le carte scoperte (non coperte da altre carte) possono essere giocate.

Una carta alla volta è scoperta nel mazzo di pesca (waste pile).

<h3>Come giocare</h3>

Puoi rimuovere una carta se il suo valore è uno in più o uno in meno rispetto alla carta corrente.

<h5>Esempi:</h5>

Se la carta corrente è 7 → puoi giocare 6 o 8

Se è Asso → puoi giocare 2 o Re

Se è Re → puoi giocare Q o Asso

<h3>Mazzo</h3>

Se non puoi fare mosse, puoi pescare una nuova carta dal mazzo.

Il mazzo può essere usato una sola volta.

<h3>Fine del gioco</h3>

Il gioco finisce quando:

tutte le carte sono rimosse → vittoria
oppure

non ci sono più mosse e il mazzo è finito → sconfitta

<h3>Facciamolo partire</h3>

Per far partire il progetto correttamente fare run da "MainGUI".
